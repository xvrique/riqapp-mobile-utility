**RiqApp** is a robust mobile application developed using Flutter. It serves as a showcase for implementing modern UI/UX design patterns, efficient state management, and seamless backend integrations.

## 🚀 Key Features
- **Intuitive UI:** Clean and responsive design tailored for a premium user experience.
- **Modular Architecture:** Built with scalability in mind using MVC
- **Real-time Integration:** Connected with REST APIs for dynamic data handling.
- **Cross-Platform:** Optimized for both Android and iOS devices.

## 🛠️ Tech Stack
- **Framework:** Flutter
- **Language:** Dart
- **State Management:** Provider / BLoC
- **Database:** SQLite
